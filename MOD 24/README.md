# MOD 24 - Colecciones Avanzadas, Enumerados y Genéricos

## Descripción del Módulo

Este módulo se enfoca en conceptos avanzados de Java SE, incluyendo el uso de colecciones avanzadas, enumerados, genéricos y mapas. Los estudiantes aprenderán a implementar sistemas complejos utilizando estas características del lenguaje Java.

## Contenidos del Módulo

### 📚 Documentación
- **DOCS/**: Contiene la documentación teórica del módulo
  - Colecciones Interfaces Queue.pdf
  - Colecciones Interfaces Set y List.pdf
  - Enumerados conceptos básicos.pdf
  - Genéricos conceptos básicos.pdf
  - Mapas conceptos básicos.pdf

### 🎯 Desafíos
- **Desafío01**: Implementación de HashSet y TreeSet con genéricos para manejo de documentos y teléfonos
- **Desafío02**: Sistema bancario con colecciones, comparadores y enumerados
- **Desafío03**: Desafío dividido en dos partes con diferentes implementaciones

### 🧪 Laboratorios
- **Laboratorio 01**: Sistema de gestión de personas con HashSet y ordenamiento
- **Laboratorio 02**: Sistema de gestión de personas con PriorityQueue para cola de atención
- **Laboratorio 03**: Sistema de gestión de personas con Mapas y utilidades avanzadas

### 📖 Material Adicional
- **Colecciones**: Ejemplos avanzados de uso de colecciones
- **Enumerados**: Implementaciones y casos de uso de enumerados
- **Genéricos**: Ejemplos de programación genérica
- **Mapas**: Manejo de mapas y estructuras de datos asociativas

## Conceptos Clave

### Colecciones Avanzadas
- HashSet y TreeSet
- PriorityQueue
- Mapas (HashMap, TreeMap)
- Interfaces List, Set, Queue, Map

### Enumerados
- Definición y uso de enum
- Métodos en enumerados
- Enumerados con valores y constructores

### Genéricos
- Programación genérica con tipos parametrizados
- Wildcards y bounded types
- Type erasure

### Comparadores y Ordenamiento
- Implementación de Comparator
- Ordenamiento de colecciones
- Comparadores personalizados

## Requisitos Previos

- Conocimiento básico de Java SE
- Comprensión de Programación Orientada a Objetos
- Familiaridad con colecciones básicas (ArrayList, LinkedList)
- Manejo de excepciones en Java

## Tecnologías Utilizadas

- Java SE
- Maven (gestión de dependencias)
- JUnit (pruebas unitarias)

## Cómo Ejecutar los Proyectos

1. Asegúrate de tener Java JDK instalado
2. Navega al directorio del proyecto deseado
3. Compila el proyecto: `mvn compile`
4. Ejecuta las pruebas: `mvn test`
5. Ejecuta la aplicación principal: `mvn exec:java -Dexec.mainClass="com.educacionIT.javase.principal.App"`

## Objetivos de Aprendizaje

Al finalizar este módulo, los estudiantes serán capaces de:

1. **Implementar colecciones avanzadas** para resolver problemas complejos
2. **Utilizar enumerados** para crear tipos de datos seguros y legibles
3. **Aplicar genéricos** para escribir código reutilizable y type-safe
4. **Manejar mapas** para estructuras de datos asociativas
5. **Implementar comparadores** para ordenamiento personalizado
6. **Diseñar sistemas** que integren múltiples conceptos avanzados de Java
